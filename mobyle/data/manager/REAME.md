# About

The pyramid application manage the upload of data for Mobyle.
User can upload multiple files and delete them if necessary.

Once user is logged, he can select among his projects the place he wants to upload his data.
It is possible to uncompress an archive once uploaded and to register the data (when uncompressed) as a single set of data.
